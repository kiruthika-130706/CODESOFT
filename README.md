# Credit Card Fraud Detection Using Machine Learning

## Project Overview

This project aims to detect fraudulent credit card transactions using Machine Learning techniques. Credit card fraud is a major issue in the financial industry, leading to significant financial losses each year. The goal of this project is to build a classification model that can accurately distinguish between legitimate and fraudulent transactions.

## Dataset

The dataset contains anonymized credit card transaction records made by European cardholders. It includes numerical features obtained through PCA transformation, along with transaction amount, time, and the target variable.

* Class 0: Legitimate Transaction
* Class 1: Fraudulent Transaction

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* KaggleHub

## Machine Learning Algorithm

* Logistic Regression

## Project Workflow

1. Download and load the dataset.
2. Perform data preprocessing and exploration.
3. Split the dataset into training and testing sets.
4. Train a Logistic Regression model.
5. Evaluate the model using:

   * Accuracy Score
   * Classification Report
   * Confusion Matrix
6. Analyze the model's performance in detecting fraudulent transactions.

## Results

The model successfully classifies credit card transactions as fraudulent or legitimate and achieves high accuracy on the test dataset. Performance is evaluated using precision, recall, F1-score, and confusion matrix metrics.

## Learning Outcomes

* Understanding fraud detection systems
* Working with highly imbalanced datasets
* Applying Logistic Regression for classification
* Evaluating machine learning models using standard metrics
* Building real-world financial security applications

## Conclusion

This project demonstrates how machine learning can be used to identify fraudulent credit card transactions and assist financial institutions in preventing fraud. The developed model provides an efficient and automated approach to transaction monitoring and risk management.
